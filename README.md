# gd4_p5cs
gd4_p5cs is an interpretation of p5.js in Godot.
This project is based on https://github.com/adcomp/Godot4_p5
gd4_p5cs adds support for C# sketches but retains support for .gd sketches

This project is designed for fast prototyping and for creating simple creative visuals. C# support makes it possible to add more advanced features through Godot's .NET ecosystem while keeping full support for .gd sketches.

Creative coding can be a great way to practice core programming fundamentals that you can apply to any project.
